Project Overview
Smart Analytics Tool is a Streamlit-based application that helps users analyze CSV datasets interactively.
Features
CSV File Upload
Dataset Preview
Missing Value Analysis
Statistical Summary
Histogram Visualization
Scatter Plot Visualization
Box Plot Visualization
Technologies Used
Python
Pandas
Streamlit
Plotly
How to Run
Open terminal in project folder
Run the command:
streamlit run app.py
Upload any CSV dataset for analysis
